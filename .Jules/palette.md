## 2024-05-22 - [Accessibility Refactor]
**Learning:** Converting interactive dashboard cards to `<button>` elements for accessibility requires applying `w-full text-left` to maintain layout and explicitly adding `cursor-pointer` for visual consistency.
**Action:** When refactoring for accessibility, always verify layout regression with visual checks, as default browser styles for buttons (center align, inline-block) often break custom layouts.
